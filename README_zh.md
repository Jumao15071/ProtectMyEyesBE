护眼大补丸是由B站up主**@黑山大叔** 制作，**@橘猫sama** 负责基岩版移植的一款数格子护眼资源包。

## 主要功能

**1. 护眼边框**
- 给多数方块增加护眼边框，方便玩家搭建筑、数格子
- 区分磨制/平滑各类砂/石台阶等方块侧面
- 区分合成台/书架/讲台等方块底面
- 改善平滑石等方块的纹理边框

**2. 红石信息显示**
- 区分投掷器/发射器/熔炉，活塞/黏性活塞
- 区分压力板/原方块，中继器/比较器底面
- 区分阳光传感器状态，提示铁砧损耗
- 显示合成器、侦测器朝向和充能情况
- 显示投掷器、发射器朝向（仅红石、功能和无缝玻璃模式）
- 显示侦测器、漏斗充能状态
- 用刷子刷红石粉可检测其信号强度
- 显示幽匿振动传播轨迹
- 显示音符盒音高和敲击次数
- 非渲染龙版本包含区块显示和红石信号显示功能
- 渲染龙版本启用着色器加载器可启用区块显示、红石信号显示等功能

*注：可使用 LeviLauncher 等第三方模组工具读取资源包内的文件，或手动改包*  
*注：矿物贴图兼容支持一部分第三方着色器下的矿物发光*  
*注：由于着色器代码变化，红区显仅支持 1.21.120，不改包则不影响使用*

**3. 清晰资源显示**
- 轮廓选择模式下光标对准的方块会有高亮白框
- 提升水雾、岩浆雾和下界环境雾的能见度
- 矿石添加清晰高亮的边框
- 启用灵动视效或光线追踪模式，矿物会发光
- 显示各类铜制方块、物品的打蜡状态
- 显示紫水晶母岩和成熟紫水晶簇
- 农作物成熟时开花，小麦成熟前更绿
- 区分西瓜茎/南瓜茎，海带不生长的顶端开花
- 区分虫蚀方块
- 区分可疑的沙子沙砾
- 低版本显示旗帜图案和刷怪蛋纹理
- 更明显的探险家地图纹理
- 快乐恶魂受伤时会显示创可贴表示低血量
- 标记烈焰人、骷髅、凋灵骷髅、僵尸猪人、岩浆怪在下界砖块上生成的位置
- 标记掠夺者在草方块、白桦木板和深色橡木木板上生成的位置
- 标记守卫者在水和海晶石上生成的位置
- 标记女巫在丛林木板上生成的位置

*注：以上位置通过粒子标记 20 分钟，退出地图后标记消失，不能 100% 还原真实生成位置（仅红石和功能模式）*

**4. 增强 UI 功能**
- 弓箭蓄力颜色提示
- 短剑和透明盾（仅红石、功能和无缝玻璃模式）
- 一键合成方块，支持背包、合成台、切石机、织布机
- 快速搬运投掷器、发射器、漏斗、各类箱子物品，选择特定物品后点击按钮可搬运特定物品
- 透视村民交易内容并一键交易
- 铁砧点击拓展按钮再编辑即可文本换行
- 手机使用结构方块可导入导出结构
- 设置界面新增快捷设置，包含常用设置选项，还包括导出模板和关闭垂直同步等特殊功能
- 添加下界、末地传送背景图以同步 Java 版
- 游戏设置添加允许命令开关
- Tab 补全指令，支持聊天框和命令方块
- 聊天框点击指令可一键补全
- 聊天框 ↑↓ 切换指令
- 为网易版添加了坐标复制和文本粘贴
- 指令补全支持网易原生魔法指令模组，优化了面板，玩家右侧屏幕点击不会退出
- 去除高版本设置面板的间隔
- 新增局内语言切换
- 修复基岩版译名文本错误（采用 wiki 译名修复包）
- 添加中国台湾、中国香港、文言文语言模式
- 以 OreUI 和纹理更新后风格为基础重绘部分 UI 背景贴图

*注：快捷设置的默认游戏模式从生存切旁观不会影响作弊*

**5. 细节优化**
- 降低折跃门、三叉戟、活塞、投掷器发射器声音，保护耳朵
- 修复活塞、箱子、TNT 爆炸声音以同步 Java 版
- 添加苦力怕脚步声，僵尸的脚步声会更明显
- 修复珊瑚、钟等方块错误音效
- 添加暴击音效以同步 Java 版
- 修复光源方块、末影龙火球等方块物品纹理以同步 Java 版
- 修复物品床、染料颜色错误
- 为方块添加 tga 贴图以提升游戏性能并改善远景模糊
- 设置界面背景改为黑色半透明，红石模式透明度进一步降低
- 改进网易独有按键和 UI 贴图，修复网易贴图 bug
- 优化原版造成卡顿的药水/信标 buff 粒子，减少生物死亡粒子，减少作物催熟粒子，移除下界传送门粒子以改善游戏性能
- 同步 Java 版 TNT 爆炸粒子效果
- 凋零状态玩家血量更明显
- 南瓜头、望远镜边缘改为半透明

*注：新版已重写实体文件，兼容 action&stuff*
